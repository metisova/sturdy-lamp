## Local setup

```bash
# clone repository
git clone git@github.com:metisova/sturdy-lamp.git
```

### Start the server

```bash
# cd into server
cd server

# install server's dependencies
npm install

# run server
npm run start:server
```

### Start the client

```bash
# open another terminal and cd into client
cd client

# install client's dependencies
npm install

# run client
npm run dev
```
